== Description ==
A WooCommerce extension that gives you an ability to display wholesale price on all products of your WooCommerce store. Admin need to assign a wholesaler customer by selecting their role as "Wholesaler" manually, wholesale prices only visible for Wholesaler role Customers not for public customers. 

In [Pro Version](https://woocommerce.com/products/wholesale-for-woocommerce/) it has Multiple user role Wholesale pricing feature, in which you can create multiple wholesale roles and define different discounts for each role with support for simple and variable products both. Also, there is a Front End Wholesale Registration form which you can customize as per your requirement.

* Translation using .PO .MO files

== Installation ==
To add a WordPress Plugin using the built-in plugin installer:

Go to Plugins > Add New.

1. Type in the name \"WooCommerce Wholesale Pricing\" in Search Plugins box
2. Find the \"WooCommerce Wholesale Pricing\" Plugin you wish to install.
3. Click Install Now to install the WordPress Plugin.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or return to plugin installer for further actions.
